# 🐶 うちの犬 お散歩ダッシュ

ブラウザで遊べる、シンプルなお散歩アクションゲームです。  
障害物をジャンプで避けながら、ステージクリアを目指します。

---

## 🎮 ゲーム概要
- ジャンル：横スクロールアクション
- プレイ環境：PC / スマホ対応（ブラウザ）
- ステージ制（全3ステージ）
  - ステージ1：公園
  - ステージ2：商店街
  - ステージ3：ドッグカフェ（高難易度）

---

## 🕹 操作方法
- **左右移動**：← → / A D
- **ジャンプ**：Space または クリック（2段ジャンプ可）
- **スタート**：画面上の「スタート」ボタン
- **リトライ**：ゲームオーバー後に可能
